# Benched Growth & SEO Roadmap
Based on targeted execution priorities for organic search and conversion scaling.

## Phase 1: Fundamentals & Tracking (Weeks 1-2) 🚀 CURRENT FOCUS
### 1.1 Technical SEO & Health
- [ ] Review Google Search Console for Excluded/Soft 404/Redirect anomalies.
- [ ] Implement tight canonical tags uniformly across dynamic filters.
- [ ] Validate `Last-Modified` headers structure for SEO crawlers.
- [ ] Run Lighthouse audits on Top 5 templates (Home, Browse, Product, Cards, Guide).
- [ ] Compress hero images and optimize LCP elements to green.
- [ ] Ensure all top-level category pages have unique `<title>` and `<h1>` tags with Meta Descriptions.
- [ ] Map internal links from deep blog/guide posts to direct money (product) pages.
- [ ] Implement FAQ blocks using structured JSON-LD data.

### 1.2 Analytics Implementation
- [ ] Finalize GA4 Configuration (`NEXT_PUBLIC_GA_MEASUREMENT_ID`).
- [ ] Establish GA4 E-commerce / Conversion event hooks for Purchases & Signups.
- [ ] Integrate PostHog for session recording & funnel drop-off analysis.

## Phase 2: Conversion Drivers (Weeks 2-4)
### 2.1 Messaging & Trust
- [ ] Tighten hero messaging - clear value proposition ("For hoopers & collectors").
- [ ] Build "Social Proof" band on homepage (seller volume, user reviews/testimonials, secure payment badges).
- [ ] Build "Benched vs. eBay/StockX" Comparison/Alternative page (High intent keyword trap).
- [ ] Audit CTAs to ensure one *primary* action per page view without competing visual weight.

### 2.2 Functional Conversion Boosters
- [ ] "Make an Offer" system for secondary market listings.
- [ ] Product waitlist / "Notify me on restock" system.
- [ ] Price Drop alert system for Favorited items.

## Phase 3: Content & Topical Authority (Weeks 4-6)
### 3.1 Core Content Clusters
- [ ] Publish 8-12 pillar/cluster articles targeting high-intent (e.g., "Best Outdoor Basketball Shoes 2026", "Investing in Panini Prizm").
- [ ] Develop dedicated landing pages for niche use-cases (e.g., "Shoes for Flat Feet", "Grading PSA/BGS guide").

## Phase 4: Programmatic Expansion (Weeks 6-10)
### 4.1 Scaling the Data Engine
- [ ] Generate unique, scalable intro blurbs for multi-variant pages so Google doesn't flag them as thin content.
- [ ] Add dynamic FAQ sections to programmatic category pages pulling real database questions.
- [ ] Add `SoftwareApplication` / `HowTo` schema to appropriate guide flows.

## Phase 5: Ongoing Cadence (Recurring)
### 5.1 Maintenance
- [ ] **Monthly:** Review up-trending queries in GSC -> Refresh existing pages to steal CTR.
- [ ] **Quarterly:** Prune, merge, or 301 redirect dead/underperforming URLs to consolidate domain authority.
